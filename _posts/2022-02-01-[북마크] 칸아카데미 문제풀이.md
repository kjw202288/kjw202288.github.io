## 원자 구조 및 속성

### 몰과 몰 질량

#### 문제유형1

| | | 
|:---|
| 문제 | 은의 몰질량 107.87 [g/mol], 1.97x10^22 [atoms]이 포함된 은의 그램을 계산하라 |
| 1 | 1.97x10^22 atoms x 1 mol / 6.022x10^23 atoms (아보가드로 수) = 3.27 x 10^-2 mol |
| 2 | 3.27 x 10^-2 mol x 107.87 g / 1 mol = 3.53 |  

#### 문제유형2

| | | 
|:---|
| 문제 | 규소의 몰질량 28.09 [g/mol], 샘플의 92.8 [mg]의 분자량 계산 |
| 1 | 92.8 mg x 1 g / 1000 mg x 1 mol / 28.09 g = 3.30 x 10^-3 mol |
| 2 | 3.30 x 10^-3 mol x 6.022 x 10^23(아보가드로 수) / 1 mol = 1.99 x 10^21 |

#### 문제유형3

| | | 
|:---|
| 문제 | 표에 나온 샘플에 포함된 33.8 [g]의 몰수단위 계산, SrF2 |
| 1 | F 19.00 [g/mol] |
|   | Sr 87.62 [g/mol] |
| + | SR -> 1 mol x 87.62 g/mol = 87.62g Sr |
|   | F -> 2 mol x 19.00 g/mol = 38.00g F |
| = | 125.62g |
| 3 | 33.8 g x 1 mol / 125.62 g = 0.269 mol |
|   | 0.269 mol x 6.022 x 10^23(아보가드로 수) / 1 mol = 1.62 x 10^23 |

#### 문제유형4

| | | 
|:---|
| 문제 | 표를 보고 C3H6O3 용액에 5.73[kg]에 들어있는 몰수를 계산하라 |
| 1 | H  1.008
|   | C  12.01
|   | O  16.00
| + | C -> 3 mol x 12.01 g/mol = 36.03 g
|   | H -> 6 mol x 1.008 g/mol = 6.048 g
|   | O -> 3 mol x 16.00 g/mol = 48.00 g
| = | 90.08g
|   | 5.73 kg x 1000 g / 1 kg x 1 mol / 90.08 = 63.6 mol |
