#### 문제유형1

은의 몰질량 107.87 [g/mol], 1.97x10^22 [atoms]이 포함된 은의 그램을 계산하라 
- 1.97x10^22 atoms x 1 mol / 6.022x10^23 atoms (아보가드로 수) = 3.27 x 10^-2 mol 
- 3.27 x 10^-2 mol x 107.87 g / 1 mol = 3.53 

#### 문제유형2

규소의 몰질량 28.09 [g/mol], 샘플의 92.8 [mg]의 분자량 계산
- 92.8 mg x 1 g / 1000 mg x 1 mol / 28.09 g = 3.30 x 10^-3 mol
- 3.30 x 10^-3 mol x 6.022 x 10^23(아보가드로 수) / 1 mol = 1.99 x 10^21

#### 문제유형3

### 순수한 물질의 원소조성

#### 문제유형 1

C(탄소) 80%, H(수소) 20% 포함하는 화합물을 구하라
- C -> 80 g x 1 mol / 12 g = 6.6 mol
- H -> 20 g x 1 mol / 1 g = 20 mol
- C -> 6.6 / 6.6 = 1 C
- H -> 20 / 6.6 = 3 H 
- CH3

#### 문제유형 2

H2O 36.0 g, CO2 66.0 g이며 탄소와 수소만 포함하여 반응하는 화합물을 고르라
- CO2 -> 66.0 g x 1 mol / 44 g CO2 x 1 mol C / 1 mol CO2 = 1.5 mol C
- H2O -> 36.0 g x 1 mol / 18 H2O x 2 mol H / 1 mol H2O = 4 mol H
- C -> 1.5 / 1.5 = 1 C x 3 = 3 C
- H -> 4 / 1.5 = 2.6 H x 3 = 8 H
- C3H8

#### 문제유형 3

118.0 g 회합물 샘플에 C(탄소) 72.0g, H(수소) 18.0g, N(질소) 28.0g 포함되어있다 화합물 실험식을 구하라
- C -> 72 g x 1 mol / 12 g = 6 mol
- H -> 18 g x 1 mol / 1 g = 18 mol
- N -> 28 g x 1 mol / 14 g = 2 mol
- C 6/2 = 3C
- H 18/2 = 9H
- N 2/2 = 1N
- C39HN
